Esta é uma **Landing Page Institucional** responsiva e moderna desenvolvida para o setor de agronegócio sustentável. O projeto foi estruturado com base em conceitos de alta produtividade, inovação tecnológica no campo e preservação ambiental ativa, ideal para a apresentação de soluções agroecológicas e agrícolas tecnológicas.

---

## 📸 Demonstração das Seções

O layout foi planejado seguindo a estrutura de 3 slides sequenciais apresentados no design original:

1. **Header & Seção Hero (Home):** Banner de tela cheia com o horizonte da plantação, título impactante em caixa alta, barra de pesquisa funcional e menu hambúrguer dinâmico.
2. **O Futuro do Campo é Sustentável:** Grid contendo duas imagens agrícolas sobrepostas com bordas arredondadas verdes e bloco de texto institucional.
3. **Inovação & Preservação:** Grid com padrão de listras verticais decorativas inspiradas em madeira/linhas de plantio, imagem de colheita mecânica e blocos de tópicos destacados.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído puramente com tecnologias web nativas, garantindo leveza, SEO otimizado e fácil manutenção:

* [**HTML5**](https://developer.mozilla.org/pt-BR/docs/Web/HTML) ─ Estruturação semântica do conteúdo.
* [**CSS3**](https://developer.mozilla.org/pt-BR/docs/Web/CSS) ─ Estilização moderna, efeitos de sobreposição de imagens, padrões lineares e responsividade (Mobile First).
* [**JavaScript (ES6+)**](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) ─ Manipulação do DOM para ativação do menu hambúrguer interativo e comportamento de scroll suave.
